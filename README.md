# ☁️ Weather Forecast
 My basic app to demonstrate newly learned technologies.
 
 # Tech Stack
 
 * Kotlin
 * MVVM 
 * ViewBinding
 * Dagger + Hilt
 * Retrofit + Gson + Callback library
 * Room
 * Offline Caching (NetworkBoundResource)
