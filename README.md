# ☁️ Weather Forecast
 My basic app to demonstrate newly learned technologies.
 
 
 # Tech Stack
 
 * Kotlin
 * MVVM 
 * ViewBinding
 * Dagger + Hilt
 * Retrofit ( 3 API's ) + Gson + Callback library
 * Room
 * Flows
